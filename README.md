# fttx.gr-api

> Backend API for FTTX.GR

## Running the app

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

```
# Install depedencies
npm install

# Run in dev mode
npm run dev

# Run in prod mode
npm start
```

The API is live at [api.fttx.gr](https://api.fttx.gr)
